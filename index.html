无法访问？修改 DNS 即可解决 解决 100% 访问问题
大多数访问失败是由 DNS 污染导致的，换用公共 DNS 可立即修复，无需任何工具。推荐使用以下服务器：
Google DNS 8.8.8.8 点击复制
Google 备用 8.8.4.4 点击复制
Cloudflare DNS 1.1.1.1 点击复制

💡 仅对当前 Wi-Fi 生效，建议在路由器统一修改。

  1. 1
    设置 → 网络与互联网 → 长按当前 Wi-Fi → 修改网络
  2. 2
    展开 高级选项，IP 设置改为 静态
  3. 3
    DNS 1 填 8.8.8.8，DNS 2 填 1.1.1.1
  4. 4
    点击 保存，重连 Wi-Fi

💡 也可搜索"私人 DNS"，填入 dns.google，对所有网络生效。

  1. 1
    浏览器访问 192.168.1.1 或 192.168.0.1 登录路由器后台
  2. 2
    找到 WAN 设置 → DNS 设置
  3. 3
    首选 DNS 填 8.8.8.8，备用填 1.1.1.1
  4. 4
    保存后路由器重启，全屋设备即时生效

💡 路由器修改一次，家中所有设备同时受益，优先推荐。
